# 🦁 SPHINX Strategy — Legendary Edition

## ما هي SPHINX؟

**SPHINX** = **S**mart **P**rice-action **H**ybrid **IN**tegrated e**X**ecution

استراتيجية ابتكار خاص تعتمد على مفهوم **"السمارت ماني"** (Smart Money Concepts):

### 🔬 المنطق العلمي

```
الخطوة 1: الترند القوي (15m EMA 21 > 55)
        ↓
الخطوة 2: مسح السيولة (Liquidity Sweep)
        السعر يكسر قاع حديث ثم يغلق فوقه
        = الحيتان جمعت مراكزها
        ↓
الخطوة 3: تأكيد الحجم (Volume > 1.5x)
        ↓
الخطوة 4: تباعد الزخم (RSI Divergence)
        السعر: قاع أدنى | RSI: قاع أعلى
        ↓
الخطوة 5: دخول فوري + SL/TP ديناميكي
```

### ⚡ مميزات SPHINX

| الميزة | الشرح |
|--------|-------|
| 🎯 **دقة عالية** | لا تدخل إلا بعد 4 تأكيدات |
| 🛡️ **SL ذكي** | أسفل القاع بـ 1x ATR — يتجنب الإيقاف المبكر |
| 💰 **TP سخي** | 2.5x المخاطرة (Risk/Reward = 1:2.5) |
| 🧠 **تفكير مثل الحيتان** | تدخل بعد "مسح السيولة" وليس قبله |
| ⏱️ **فريم 15 دقيقة** | توازن بين السرعة والدقة |
